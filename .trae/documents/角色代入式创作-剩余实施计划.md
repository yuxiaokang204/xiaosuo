## 概述

## 关键问题诊断

### 问题1：角色代入被跳过（P0 阻断）
- **位置**: `chapter_pipeline.py:471` `_get_active_characters_for_chapter`
- **原因**: 该方法读取 `self._current_context.get("characters_list", [])`，但 orchestrator 的 `shared_context` 只传了 `"characters"`（JSON字符串），没有 `"characters_list"`（列表）
- **后果**: `active_characters` 永远为空 → 角色代入步骤被跳过 → 整个v6.0机制失效
- **修复**: orchestrator 在 `shared_context` 中新增 `"characters_list"` 字段（原始列表）

### 问题2：story_direction未生成（P0）
- **位置**: `orchestrator.py` run_stage/run_all_loop
- **原因**: 没有生成故事走向的阶段，`shared_context` 中无 `story_direction`
- **后果**: `_agent_draft` 拿到空的story_direction，角色代入卡也拿不到故事走向
- **修复**: 在 outlining 阶段后（或 Loop 0 S5 之后）新增 story_direction 生成，写入 state 和 shared_context

### 问题3：StateTracker.reset() 遗漏（P0）
- **位置**: `state_tracker.py:583` reset()
- **原因**: 新增了 `_character_memories` 字段，但 reset() 未清空
- **后果**: 新小说开始时残留上一本小说的角色记忆
- **修复**: reset() 中添加 `self._character_memories.clear()`

### 问题4：章节后未更新记忆（P0）
- **位置**: `chapter_pipeline.py` run() 方法
- **原因**: `_update_character_memories_after_chapter` 方法已实现但未在 run() 中调用
- **后果**: 角色经历记忆链永远为空，角色代入卡拿不到记忆影响
- **修复**: 在 run() 的记忆更新区块后调用 `_update_character_memories_after_chapter`

### 问题5：Loop 0并行角色生成缺世界观（P1）
- **位置**: `orchestrator.py:926` `_do_char_skeleton`
- **原因**: Loop 0 中世界观和角色并行生成，角色生成时 `world_info=""` 为空
- **后果**: 角色背景无法嵌入世界观设定
- **修复**: Loop 0 改为串行（先世界观后角色），Loop 1+ 保持并行（已有世界观）

#### 3.2 新增 story_direction 生成方法

```python
async def _generate_story_direction(self) -> str:
    """v6.0 基于标题/类型/风格生成故事走向"""
    from ..agents.prompts import STORY_DIRECTION_SYSTEM_PROMPT, build_story_direction_user_prompt
    from ..llm.client import LLMMessage, get_default_llm_client
    try:
        client = get_default_llm_client()
        user_prompt = build_story_direction_user_prompt(
            title=self.state.title,
            theme=self.state.theme,
            tone=self.state.tone,
            platform=self.state.platform,
            chapter_count=self.state.chapter_count,
            world_info=self._stringify(self.state.world_settings)[:500],
        )
        result = await client.generate(
            [LLMMessage(role="user", content=user_prompt)],
            system_prompt=STORY_DIRECTION_SYSTEM_PROMPT,
            temperature=0.7,
            max_tokens=800,
        )
        direction = (result.content or "").strip()
        if direction:
            self.state.story_direction = direction
            await self._emit("story_direction", {"data": direction})
            return direction
    except Exception as e:
        print(f"[Orchestrator] ⚠️ 故事走向生成失败: {e}")
    return ""
```

### 任务6：consistency_checker 新增角色代入一致性校验 [P1]
**文件**: `src/backend/core/consistency_checker.py`
**改动**: 新增方法 `check_roleplay_consistency`，校验：
1. 角色行为是否符合其代入卡的行为倾向
2. 角色是否"失忆"（提到角色不应该知道的信息）
3. 角色情绪状态是否与经历记忆链一致
4. 角色对话风格是否符合语言指纹

在 `check_chapter` 中调用（可选，depth>=1时）：
```python
def check_roleplay_consistency(self, chapter_idx: int, content: str,
                                characters: List[Dict],
                                character_memories: Dict[str, List] = None) -> List[Dict]:
    """v6.0 角色代入一致性校验"""
    issues = []
    for char in characters:
        name = char.get("name", "")
        if not name or name not in content:
            continue
        # 1. 检查角色是否提到不该知道的信息
        memories = (character_memories or {}).get(name, [])
        known_info = set()
        for m in memories:
            for info in (m.information_gained if hasattr(m, 'information_gained') else []):
                known_info.add(info)
        # 2. 检查语言指纹一致性
        speech = char.get("speech_fingerprint", {})
        if isinstance(speech, dict):
            taboo = speech.get("taboo_words", [])
            if isinstance(taboo, list):
                for tw in taboo:
                    if tw and tw in content:
                        idx = content.find(tw)
                        context_before = content[max(0, idx-100):idx]
                        if name in context_before:
                            issues.append({
                                "type": "character_ooc",
                                "severity": "warning",
                                "chapter": chapter_idx,
                                "description": f"角色'{name}'使用了禁用词'{tw}'，违反语言指纹",
                                "suggestion": "检查角色对话是否符合其语言指纹设定",
                            })
        # 3. 检查行为标签一致性
        behavior_tags = char.get("behavior_tags", [])
        if isinstance(behavior_tags, list):
            for tag in behavior_tags:
                # 简化检查：行为标签应在文中有所体现
                pass
    return issues
```
